# National Financial Modeling Engine & Commission Payout Matrix

## 📊 Business Context & Challenge
The executive leadership team at the Ibadan headquarters lacked automated, immediate visibility into cross-border financial health, state-by-state performance margins, and variable multi-tiered agent commission metrics. Because transaction histories were logged on independent local workbooks, manually calculating split payout percentages (70% Agent / 30% Agency split) and evaluating total regional sales against fixed corporate annual targets required days of administrative slicing.

## 🛠️ Advanced Data Mechanics Applied
*   **Cross-Sheet Dynamic Arrays:** Deployed nested `XLOOKUP` matrices to map individual closed sales figures from the national ledger directly against centralized performance mapping lists and local director assignments.
*   **Multi-Criteria Array Logic:** Programmed conditional strings utilizing `SUMIFS`, `COUNTIFS`, and `IFS` functions to instantly calculate state-by-state actual earnings compared against corporate baseline projections.
*   **Executive Business Intelligence Panel:** Built an interactive corporate dashboard using interconnected **Pivot Tables, customized Slicers, and Timelines**, giving senior leadership the power to toggle and inspect company sales data, field agent payouts, and state target margins in under three seconds.

## 🎯 Production Deliverable
The final product is an Interactive Executive Financial Matrix (`Oshad_Group_National_Revenue_And_Payout_Engine.xlsx`). It contains dynamic reference tabs that automatically sync regional targets with active ledger lines, driven by user-controlled slicers for instantaneous multi-variant state filtering.
